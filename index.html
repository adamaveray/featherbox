Featherbox

A lightweight, simple to customise modal/lightbox jQuery plugin with arguably aesthetically-pleasing default styles.

  • Modern styles
  • Customisable
  • IE9-compatible

Example Modal

This is an example HTML modal. Any content can be added to a modal, including images, videos and iframes.

Featherbox provides the wrapping modal styles and infrastructure, the contents of the modal are left to be styled by the user. For example, this modal is styled with roughly 70 lines of CSS.

See the Github repository to download Featherbox or view more examples and full documentation.
